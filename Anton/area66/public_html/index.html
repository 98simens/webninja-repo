Vill du ha lösningen för Företagsfesten, Barnkalaset, Svensexan, Möhippan eller om du vill lira en helt vanlig kväll?

Hos oss kan du bland annat spela Lasergame, vilket passar nästan alla åldrar. Vi rekommenderar däremot att de som spelar är över 6 år gamla.

Vi använder oss av ett system som heter VEQTOR MAXblaster vilket är ett radiobaserat system. Det tillåter gästerna som sitter i cafeterian att se hur poängen förändras för de spelande.
